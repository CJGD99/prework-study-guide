Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are referred to as 'child elements'.
  • Elements with other elements nested within them are referred to as 'parent elements'.
  • The 'h' tag creates a header element on the page, ranging by importance from 1-6.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicats how much space we want around the content inside an element
  • The CSS box model defines how the browser will lay out our elements on a webpage using space for padding, margin, border and content.
  • We can define colors on our webpage using multiple attributes such as: semantic CSS (color names), hexadecimal codes, and rgb values.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to create interactivity on a webpage
